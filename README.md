# Comp322Labs

## Lab 0
to build on Windows

### Building
```
$ cd Labs/zero
$ mingw32-make
```
to build on Unix: 
```
$ cd Labs/zero
$ make 
```
### Run

You can pass binary through stdin 
``` 
$ cd Labs/zero
$ run 11111111 00000000 11111111
``` 

Or pass in a file
```
$ cd Labs/zero
$ run - data.txt
```
